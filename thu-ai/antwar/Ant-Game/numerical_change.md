# 防御塔数值改动说明

本次修改按照 `README.md` 中“防御塔数据列表”里用 `->` 标记的条目执行，含义均为“旧值 -> 新值”。

## 改动列表

- `Heavy+`
  - 伤害：`25 -> 24`
  - 影响：单次点杀能力略降，仍保持重型单体高伤定位。

- `Ice`
  - 伤害：`8 -> 12`
  - 范围：`1 -> 2`
  - 影响：冻结塔从近距离单点干扰改为更强的中近程控制点，覆盖面和补伤能力都上升。

- `Bewitch`
  - 伤害：`8 -> 14`
  - 影响：蛊惑塔在保留控制效果的同时，补足了命中后的直接伤害压力。

- `Mortar`
  - 伤害：`8 -> 12`
  - 影响：一级迫击炮的范围压制能力提升，更适合处理中密度蚁群。

- `Mortar+`
  - 伤害：`15 -> 18`
  - 影响：二级迫击炮的 AOE 斩杀线提高，强化中后期群伤压制。

- `Pulse`
  - 伤害：`8 -> 14`
  - 影响：脉冲塔在群体转随机之外，具备更明显的清场和压血能力。

- `Missile`
  - 伤害：`15 -> 18`
  - 影响：导弹塔的大范围爆发进一步加强，对后排和密集目标的威胁更高。

## 未改动项

- 上述塔种的攻击间隔、生命值、特殊效果说明保持不变。
- `Ice` 之外，其余本次带 `->` 标记的塔的攻击范围未变化。
